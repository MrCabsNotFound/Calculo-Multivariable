# Proyecto Final - Cálculo Multivariable

## Contexto
Proyecto universitario de la clase de Cálculo Multivariable, UFM.
El código debe parecer escrito por estudiantes universitarios.

## Estilo de código obligatorio
- Cada línea de código debe tener un comentario con #
- Usar las mismas librerías vistas en clase: NumPy, Matplotlib, SymPy
- Celdas simples y directas, sin sobre-ingeniería
- Sin clases (OOP) innecesarias, preferir funciones simples
- Outputs con print() simples o tablas con tabulate si aplica
- En español, tanto comentarios como outputs al usuario

## Qué NO hacer
- No usar librerías que no aparecen en los notebooks de clase
- No hacer el código más sofisticado de lo necesario
- No agregar manejo de errores complejo
- No usar type hints ni decoradores
